# 🛡️ Project Inbox Defender: Interactive Phishing Awareness Platform

An interactive, multi-level simulator designed to train users on identifying sophisticated phishing attacks and social engineering tactics. This project transitions traditional, passive security training into an engaging, gamified defense grid.

## 🎯 Project Objective
This platform was engineered to satisfy the core objectives of **Task 2: Phishing Awareness Training**:
* **Interactive Module Delivery:** Replacing static presentations with a functional web application.
* **Deception Training:** Teaching users to recognize malicious URLs, spoofed senders, and fake domains.
* **Psychological Analysis:** Educating users on social engineering behavioral hooks (Urgency, Authority, and Scarcity).
* **Real-World Scenarios:** Deploying complex, data-driven attack scenarios modeled after industry incidents.
